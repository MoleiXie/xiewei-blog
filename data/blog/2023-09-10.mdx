## 介绍
子元素 flex 包含以下三个属性
- `flex-glow` 占据剩余空间的比率
- `flex-shrink` 空间不足时收缩的比率
- `flex-basis` 可以理解为在容器中的宽度(同时设置width和这个属性时，该属性具备更高的优先级)

## 作用
常见的 flex: 1 相当于占据了容器的剩余空间，不具备固定大小<br/>
当容器扩大时元素也随之变大，容器缩小时也随之缩小
```css
/* box 与 box1 完全相当 */ 
.box{
    flex: 1;
}
/* 当设置flex: 1时, 就相当于设置了以下属性值 */
.box1{
    flex-glow: 1;   // 表示占据所有的剩余空间
    flex-shrink: 1; // 空间不足时, 进行收缩
    flex-basis: 0%; // 自动填充剩余所有空间
}
```

## 示例
这是一个非常经典的三栏布局，左右200px固定，中间自适应。<br/>
中间部分即flex：1，空间收缩时会随之收缩，增大时会随之增大。

<div className="flex">
    <div className="w-[200px] p-4 bg-gray-200">Left Sidebar</div>
    <div className="flex-auto p-4 bg-gray-400">Main Content</div>
    <div className="w-[200px] p-4 bg-gray-200">Right Sidebar</div>
</div>
```jsx
<div className="flex">
    <div className="w-[200px] p-4 bg-gray-200">Left Sidebar</div>
    <div className="flex-auto p-4 bg-gray-400">Main Content</div>
    <div className="w-[200px] p-4 bg-gray-200">Right Sidebar</div>
</div>
```
