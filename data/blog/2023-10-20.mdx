# MDX介绍

MDX 是 markdown、JSX、Javascript表达式以及ESM中的`export`和`import`语句的集合体<br/>
可以理解为在markdown中嵌入JSX语法，让组件可以在文档中使用

# markdown 语法基础

## 1. 标题
```
# 标题一

## 标题二

#### 标题四
```
# This is a h1 tag

## This is a h2 tag

#### This is a h4 tag<br/>


## 2. 强调
```
_这是斜体文本_

**这是加粗文本**

_斜体 **结合** 加粗文本_

~~这是删除线~~
```
_这是斜体文本_

**这是加粗文本**

_斜体 **结合** 加粗文本_

~~这是删除线~~

## 3. 列表

### 无序列表

```
- Item 1
- Item 2
  - Item 2a
  - Item 2b
```

- Item 1
- Item 2
  - Item 2a
  - Item 2b

### 有序列表

```
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   1. Item 3b
```

1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   1. Item 3b

## 3. 图片

```
语法格式: ![Alt Text](url)

示例：![photos](https://images.pexels.com/photos/2559941/pexels-photo-2559941.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1)
```

## 4. 跳转链接

```
语法格式: [link Text](url)

示例：跳转到 [@GitHub](http://github.com)
```
跳转到 [@GitHub](http://github.com)

```
语法格式：> 引用文字xxxxxx
```

```
语法格式：`line code`
```

## 9. 脚注

```
会在文章结尾生成[^1].

[^1]: 脚注1.
```

## 10. 任务列表

```
- [x] list syntax required (any unordered or ordered list supported)
- [x] this is a complete item
- [ ] this is an incomplete item
```

- [x] list syntax required (any unordered or ordered list supported)
